11. If X and Y are independent random variables with variances 2 and 3. Find the variance of 3X + 4Y.

Given: \(Var(X) = 2\), \(Var(Y) = 3\). X and Y are independent.

We need to find \(Var(3X + 4Y)\).

For independent X and Y, \(Var(aX + bY) = a^2 Var(X) + b^2 Var(Y)\).

\(Var(3X + 4Y) = 3^2 Var(X) + 4^2 Var(Y) = 9(2) + 16(3) = 18 + 48 = 66\)

Answer: The variance of 3X + 4Y is 66.

12. If the density function of a continuous random variable X is given by

\(f(x) = \begin{cases} ax, & 0 \le x \le 1 \\ a, & 1 \le x \le 2 \\ 3a-ax, & 2 \le x \le 3 \\ 0, & \text{otherwise} \end{cases}\)

find a.

For \(f(x)\) to be a PDF, \(\int_{-\infty}^{\infty} f(x) dx = 1\).

\(\int_0^1 ax \,dx + \int_1^2 a \,dx + \int_2^3 (3a-ax) \,dx = 1\)

\(a \left[ \frac{x^2}{2} \right]_0^1 + a [x]_1^2 + \left[ 3ax - a\frac{x^2}{2} \right]_2^3 = 1\)

\(\frac{a}{2} + a + (9a - \frac{9a}{2}) - (6a - 2a) = 1\)

\(\frac{a}{2} + a + \frac{9a}{2} - 4a = 1 \implies \frac{a+2a+9a-8a}{2} = 1 \implies \frac{4a}{2} = 1 \implies 2a = 1 \implies a = \frac{1}{2}\)

Answer: The value of a is \(\frac{1}{2}\).

13. The mean and variance of binomial distribution are 5 and 4. Determine the probability distribution.

Mean \(\mu = np = 5\). Variance \(\sigma^2 = np(1-p) = 4\).

\(5(1-p) = 4 \implies 1-p = \frac{4}{5} \implies p = \frac{1}{5}\).

\(n(\frac{1}{5}) = 5 \implies n = 25\).

PMF: \(P(X=k) = \binom{25}{k} \left(\frac{1}{5}\right)^k \left(\frac{4}{5}\right)^{25-k}\), for \(k = 0, 1, \dots, 25\).

Answer: \(P(X=k) = \binom{25}{k} (\frac{1}{5})^k (\frac{4}{5})^{25-k}\).

14. A typist types 2 letters erroneously for every 100 letters. What is the probability that the tenth letter typed is the first letter with an error?

Probability of error \(p = \frac{2}{100} = 0.02\). Probability of correct \(q = 1-p = 0.98\).

Geometric distribution: \(P(X=k) = q^{k-1}p\).

\(P(X=10) = (0.98)^{9} (0.02) \approx 0.83374776 \times 0.02 \approx 0.0167\).

Answer: \(\approx 0.0167\).

15. The moment generating function of a random variable X is given by \(M(t) = e^{3(e^t-1)}\). What is \(P[X=0]\)?

This is MGF of Poisson distribution with \(\lambda = 3\).

PMF: \(P(X=k) = \frac{e^{-\lambda} \lambda^k}{k!}\).

\(P(X=0) = \frac{e^{-3} 3^0}{0!} = e^{-3} \approx 0.0498\).

Answer: \(e^{-3} \approx 0.0498\).

16. A random variable X is uniformly distributed between 3 and 15. Find Var(X).

For \(U(a,b)\), \(Var(X) = \frac{(b-a)^2}{12}\).

\(Var(X) = \frac{(15-3)^2}{12} = \frac{12^2}{12} = 12\).

Answer: Var(X) = 12.

17. The cumulative distribution function of a continuous random variable is given by

\(F(x) = \begin{cases} 0, & x < 0 \\ 1-e^{-x}, & 0 \le x < \infty \end{cases}\)

Find the pdf and mean of X.

PDF \(f(x) = \frac{dF(x)}{dx} = e^{-x}\) for \(x \ge 0\), (0 otherwise).

Mean \(E[X] = \int_0^{\infty} x e^{-x} dx = 1\).

Answer: pdf \(f(x) = e^{-x}\) for \(x \ge 0\). Mean \(E[X] = 1\).

18. The joint pdf of the random variable (X, Y) is \(f(x,y) = k(x+y), 0 < x < 2; 0 < y < 2\). Find the value of k.

\(\int_0^2 \int_0^2 k(x+y) \,dx \,dy = 1 \implies k \int_0^2 (2+2y) \,dy = 1 \implies k[2y+y^2]_0^2 = 1 \implies 8k=1 \implies k = \frac{1}{8}\).

Answer: \(k = \frac{1}{8}\).

19. The joint p.d.f of the two-dimensional random variable (X, Y) is given by \(f(x,y) = kxye^{-(x^2+y^2)}, x > 0, y > 0\). Find i) k. ii) Prove that X and Y are independent.

i) \(k \left( \int_0^{\infty} x e^{-x^2} \,dx \right) \left( \int_0^{\infty} y e^{-y^2} \,dy \right) = 1\). Each integral is \(1/2\). So \(k(1/4)=1 \implies k=4\).

ii) \(f_X(x) = 2xe^{-x^2}\), \(f_Y(y) = 2ye^{-y^2}\). \(f_X(x)f_Y(y) = 4xye^{-(x^2+y^2)} = f(x,y)\).

Answer: i) \(k=4\). ii) X and Y are independent.

20. Show that \(f(x,y) = \frac{2}{5}(2x+3y), 0 \le x \le 1, 0 \le y \le 1\) is a joint pdf of X and Y.

1. \(f(x,y) \ge 0\) in the domain. 2. \(\int_0^1 \int_0^1 \frac{2}{5}(2x+3y) \,dx \,dy = \frac{2}{5} \int_0^1 (1+3y) \,dy = \frac{2}{5} [y+\frac{3y^2}{2}]_0^1 = \frac{2}{5}(\frac{5}{2}) = 1\).

Answer: \(f(x,y)\) is a joint PDF.

6. Write the properties of Correlation Coefficient.

Let \(\rho(X,Y)\) be the correlation coefficient.

  1. \(-1 \le \rho(X,Y) \le 1\).
  2. If \(\rho(X,Y) = 1\), perfect positive linear relationship.
  3. If \(\rho(X,Y) = -1\), perfect negative linear relationship.
  4. If \(\rho(X,Y) = 0\), X and Y are uncorrelated.
  5. If X and Y are independent, then \(\rho(X,Y) = 0\). Converse not always true.
  6. \(\rho(X,Y) = \rho(Y,X)\).
  7. \(\rho(aX+b, cY+d) = \frac{ac}{|ac|} \rho(X,Y)\) for \(a,c \neq 0\).
7. State Central Limit Theorem for independent identically distributed random variables.

Let \(X_1, \dots, X_n\) be i.i.d. random variables with mean \(\mu\) and variance \(\sigma^2\). Let \(S_n = \sum X_i\). Then as \(n \to \infty\), \(Z_n = \frac{S_n - n\mu}{\sigma\sqrt{n}}\) converges in distribution to \(N(0,1)\).

8. Define a strictly stationary random process.

A random process \(\{X(t)\}\) is strictly stationary if for any \(t_1, \dots, t_n\) and any \(h\), the joint CDF of \(X(t_1), \dots, X(t_n)\) is the same as the joint CDF of \(X(t_1+h), \dots, X(t_n+h)\).

\(F_{X(t_1), \dots, X(t_n)}(x_1, \dots, x_n) = F_{X(t_1+h), \dots, X(t_n+h)}(x_1, \dots, x_n)\).

9. Define Cross Correlation.

For real random processes \(X(t)\) and \(Y(t)\), the cross-correlation function is \(R_{XY}(t_1, t_2) = E[X(t_1)Y(t_2)]\). If jointly WSS, \(R_{XY}(\tau) = E[X(t)Y(t+\tau)]\).

10. Find the variance of the stationary process \(\{X(t)\}\) whose auto correlation function is given by \(R_{XX}(\tau) = 2+4e^{-2|\tau|}\).

\(Var(X(t)) = R_{XX}(0) - (E[X(t)])^2\). \(R_{XX}(0) = 2+4e^0 = 6\).

\((E[X(t)])^2 = \lim_{|\tau|\to\infty} R_{XX}(\tau) = 2\).

\(Var(X(t)) = 6 - 2 = 4\).

Answer: Variance = 4.

11. State any two properties of the power spectral density function.

Let \(S_{XX}(\omega)\) be the PSD.

  1. \(S_{XX}(\omega) \ge 0\) (Non-negativity).
  2. For real WSS process, \(S_{XX}(\omega) = S_{XX}(-\omega)\) (Even function).
  3. \(R_{XX}(0) = \frac{1}{2\pi} \int_{-\infty}^{\infty} S_{XX}(\omega) d\omega\) (Total average power).

13. a) Show that \(X(t) = A \sin(\omega t + \theta)\) is WSS if A, \(\omega\) are constants and \(\theta \sim U(0, 2\pi)\).

1. \(E[X(t)] = A E[\sin(\omega t + \theta)] = A \int_0^{2\pi} \sin(\omega t + \theta) \frac{1}{2\pi} d\theta = \frac{A}{2\pi} [-\cos(\omega t + \theta)]_0^{2\pi} = 0\) (constant).

2. \(R_{XX}(t_1, t_2) = E[A^2 \sin(\omega t_1 + \theta)\sin(\omega t_2 + \theta)]\)

\(= \frac{A^2}{2} E[\cos(\omega(t_1-t_2)) - \cos(\omega(t_1+t_2)+2\theta)]\).

\(E[\cos(\omega(t_1+t_2)+2\theta)] = \int_0^{2\pi} \cos(\omega(t_1+t_2)+2\theta) \frac{1}{2\pi} d\theta = 0\).

\(R_{XX}(t_1, t_2) = \frac{A^2}{2} \cos(\omega(t_1-t_2))\). Depends only on \(\tau = t_1-t_2\).

Answer: The process is WSS.

14. a) \(X(t) = A \cos(\omega t + \theta)\), \(Y(t) = A \sin(\omega t + \theta)\). Find Cross-Correlation coefficient function.

\(R_{XY}(t_1, t_2) = E[A^2 \cos(\omega t_1 + \theta)\sin(\omega t_2 + \theta)]\)

\(= \frac{A^2}{2} E[\sin(\omega(t_1+t_2)+2\theta) - \sin(\omega(t_1-t_2))]\).

\(E[\sin(\omega(t_1+t_2)+2\theta)] = 0\).

\(R_{XY}(t_1, t_2) = -\frac{A^2}{2} \sin(\omega(t_1-t_2))\). Let \(\tau = t_1-t_2\), \(R_{XY}(\tau) = -\frac{A^2}{2}\sin(\omega\tau)\).

\(R_{XX}(0) = A^2/2\), \(R_{YY}(0) = A^2/2\).

\(\rho_{XY}(\tau) = \frac{R_{XY}(\tau)}{\sqrt{R_{XX}(0)R_{YY}(0)}} = \frac{-\frac{A^2}{2}\sin(\omega\tau)}{A^2/2} = -\sin(\omega\tau)\).

14. b) (Or) PSD \(S(\omega) = \begin{cases} \frac{b}{a}(a-|\omega|), & |\omega| \le a \\ 0, & |\omega| > a \end{cases}\). Find autocorrelation.

\(R(\tau) = \frac{1}{2\pi} \int_{-a}^{a} \frac{b}{a}(a-|\omega|) e^{j\omega\tau} d\omega = \frac{b}{\pi a} \int_0^a (a-\omega)\cos(\omega\tau) d\omega\).

Answer: \(R(\tau) = \frac{ba}{2\pi} \text{sinc}^2(a\tau/2)\).

6. Define Markov process.

A Markov process is a stochastic process where the future state depends only on the present state, not on past states (memoryless property).

\(P(X(t_n) \le x_n | X(t_{n-1})=x_{n-1}, \dots, X(t_1)=x_1) = P(X(t_n) \le x_n | X(t_{n-1})=x_{n-1})\).

7. Let \(A = \begin{pmatrix} 0 & 1 \\ 1/2 & 1/2 \end{pmatrix}\) be a stochastic matrix. Check whether it is regular.

\(A^2 = \begin{pmatrix} 0 & 1 \\ 1/2 & 1/2 \end{pmatrix} \begin{pmatrix} 0 & 1 \\ 1/2 & 1/2 \end{pmatrix} = \begin{pmatrix} 1/2 & 1/2 \\ 1/4 & 3/4 \end{pmatrix}\). All entries are positive.

Answer: A is regular.

8. State any two properties of a Poisson process.
  1. \(N(0)=0\).
  2. Independent increments.
  3. Number of events in interval \(\tau\) is Poisson with mean \(\lambda\tau\).
  4. Inter-arrival times are i.i.d. exponential with mean \(1/\lambda\).
9. Define time invariant system.

A system \(\mathcal{T}\) is time-invariant if a time shift in the input causes an identical time shift in the output: \(\mathcal{T}[x(t-t_0)] = y(t-t_0)\), where \(y(t) = \mathcal{T}[x(t)]\).

10. Check whether the system \(y(t) = x^2(t)\) is linear.

Check homogeneity: \(\mathcal{T}[ax(t)] = (ax(t))^2 = a^2x^2(t)\). \(a\mathcal{T}[x(t)] = ax^2(t)\).

\(a^2x^2(t) \neq ax^2(t)\) for all \(a\).

Answer: Not linear.

11. If the system has impulse response \(h(t) = \begin{cases} \frac{1}{2c}, & |t| \le c \\ 0, & |t| > c \end{cases}\). Write relation between input/output spectrums.

\(H(\omega) = \mathcal{F}\{h(t)\} = \int_{-c}^{c} \frac{1}{2c} e^{-j\omega t} dt = \frac{\sin(\omega c)}{\omega c}\).

If spectrums are Fourier Transforms: \(Y(\omega) = H(\omega)X(\omega) = \frac{\sin(\omega c)}{\omega c} X(\omega)\).

If spectrums are PSDs: \(S_{YY}(\omega) = |H(\omega)|^2 S_{XX}(\omega) = \left(\frac{\sin(\omega c)}{\omega c}\right)^2 S_{XX}(\omega)\).

Relations as above.

14. a) (ii) Input to LTI system is WSS, prove output is WSS.

1. \(E[Y(t)] = E[\int h(u)X(t-u)du] = \mu_X \int h(u)du = \mu_X H(0)\) (constant).

2. \(R_{YY}(t, t+\tau) = \iint h(u)h^*(v)R_{XX}(\tau-v+u)dudv\). This depends only on \(\tau\), not \(t\).

Proof complete. Output is WSS.

14. b) (Or) (i) Prove sum of two independent Poisson processes is Poisson.

Let \(N(t) = N_1(t)+N_2(t)\) with rates \(\lambda_1, \lambda_2\).

1. \(N(0)=N_1(0)+N_2(0)=0\).

2. Increments of \(N(t)\) are sums of independent increments from \(N_1(t)\) and \(N_2(t)\), so they are independent.

3. \(N(\tau) = N_1(\tau)+N_2(\tau)\). Sum of independent Poisson RVs is Poisson with mean \((\lambda_1+\lambda_2)\tau\).

Proof complete. Sum is Poisson with rate \(\lambda_1+\lambda_2\).

14. b) (Or) (ii) Band-limited white noise \(S_{NN}(\omega) = N_0/2\) for \(|\omega \pm \omega_0| < \omega_B\). Find autocorrelation.

\(R_{NN}(\tau) = \frac{1}{2\pi} \int S_{NN}(\omega)e^{j\omega\tau}d\omega\)

\(= \frac{N_0}{4\pi} \left( \int_{-\omega_0-\omega_B}^{-\omega_0+\omega_B} e^{j\omega\tau}d\omega + \int_{\omega_0-\omega_B}^{\omega_0+\omega_B} e^{j\omega\tau}d\omega \right)\)

\(= \frac{N_0}{4\pi} \left( e^{-j\omega_0\tau}\frac{2\sin(\omega_B\tau)}{\tau} + e^{j\omega_0\tau}\frac{2\sin(\omega_B\tau)}{\tau} \right)\)

\(= \frac{N_0}{4\pi} \frac{2\sin(\omega_B\tau)}{\tau} (2\cos(\omega_0\tau)) = \frac{N_0}{\pi\tau}\cos(\omega_0\tau)\sin(\omega_B\tau)\)

\(= \frac{N_0\omega_B}{\pi} \cos(\omega_0\tau) \text{sinc}_N(\omega_B\tau)\).

Answer: \(R_{NN}(\tau) = \frac{N_0\omega_B}{\pi} \cos(\omega_0\tau) \text{sinc}_N(\omega_B\tau)\).
